# Деплой на Replit - Пошаговая инструкция

## Шаг 1: Создание проекта на Replit
1. Зайди на https://replit.com
2. Нажми "Create Repl" → "Import from GitHub"
3. Введи URL репозитория: `https://github.com/ilanmace-crypto/paradise-shop.git`
4. Выбери язык "Node.js"
5. Нажми "Import Repl"

## Шаг 2: Настройка проекта
1. В файле `package.json` измени основной скрипт:
   ```json
   "scripts": {
     "start": "node server-replit.js"
   }
   ```

2. Установи зависимости:
   - В Shell (внизу) введи: `npm install sqlite3 express cors body-parser dotenv`

## Шаг 3: Настройка .replit
1. Убедись что файл `.replit` содержит:
   ```
   language = "nodejs"
   run = "node server-replit.js"
   [env]
   PORT = "3000"
   ```

## Шаг 4: Сборка React приложения
1. В Shell введи:
   ```bash
   npm run build
   ```

## Шаг 5: Запуск проекта
1. Нажми большую зеленую кнопку "Run"
2. Сайт откроется во встроенном браузере
3. Внешний адрес будет вверху: `https://project-name.username.repl.co`

## Шаг 6: Проверка работы
1. API: `https://project-name.username.repl.co/api/health`
2. Товары: `https://project-name.username.repl.co/api/products`
3. Фронтенд: `https://project-name.username.repl.co`

## Преимущества Replit:
- **Полностью бесплатно**
- **SQLite работает из коробки**
- **Онлайн редактор** кода
- **Мгновенный запуск**
- **Автосохранение**
- **GitHub интеграция**

## Ограничения:
- **Проект спит** через 15 минут неактивности
- **Просыпается** при первом запросе
- **Ограничения** на CPU и память
- **Нет доменов** (только .repl.co)

## Если что-то не работает:
1. Проверь консоль в Shell
2. Перезапусти проект (кнопка Stop → Run)
3. Проверь что все зависимости установлены
4. Убедись что порт 3000 используется

## Готово!
Твой проект будет доступен на Replit с бэкендом и фронтендом вместе!
